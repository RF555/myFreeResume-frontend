# myFreeResume Frontend

React SPA for creating job-specific resumes and cover letters.

## Setup

```bash
npm install
```

Create a `.env` file:
```
VITE_API_URL=http://localhost:8000
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Test

```bash
npx vitest run
```

## Tech Stack

- React 18 + Vite
- Tailwind CSS + shadcn/ui
- React Query (TanStack Query)
- React Hook Form
- React Router v6
